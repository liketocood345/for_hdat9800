# Anti-patterns

| Anti-pattern | Why bad |
|--------------|---------|
| Concatenate four `SKILL.md` into one file | Wastes triggers; parallel checklists collide |
| Waist = single mega-enum | Combination explosion; belt unexpressible |
| Positive tree labels without leaf pointers | Agent stops at names; no deliverable |
| No mutex / no default | Multi-entry = random route |
| Force binary tree for 10+ lookup values | Use lookup table (gallery plot_type) |
| Load all upstream skills every turn | Context bloat; defeats fusion |
| Silent fork between allowed alternatives | Must BIND + log assumption |
| Belt in inverse tree sequence | Path explosion; duplicate rules |
| Treat diagram as sole agent spec | Drift from registry; false confidence |
| Copy upstream templates into fused skill | Breaks link-back; stale on upstream update |
